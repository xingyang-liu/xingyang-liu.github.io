LeetCode笔记1：#4 Median of Two Sorted Arrays

Description

There are two sorted arrays nums1 and nums2 of size m and n respectively.

Find the median of the two sorted arrays. The overall run time complexity should be O(log (m+n)).

中文描述

简单来说就是有两个已经排序好的数列nums1和nums2，长度分别为m和n。我们要做的是在O(log (m+n))的时间复杂度之内找出这两个数列合起来之后的中位数。

Solution

Idea

相信要找出他们的中位数并不难，难的是如何在对数复杂度的情况下找到中位数。很多人，比如我，就因为陷入了如何分割数列被各种边界条件折磨得痛不欲生，却没有想到中位数真正的作用。

In statistics, the median is used for dividing a set into two equal length subsets, that one subset is always greater than the other

不管数列长度是奇数还是偶数，中位数都是为了将一个数列划分为两个长度相等的子数列，且一个子数列的所有成员都大于另一个。

因此，我们首先随机在数列A中找到一个位置i, (i ∈ [0,m]);在数列B中找到一个位置j, (j ∈ [0,n]).

      left_part          |        right_part
A[0], A[1], ..., A[i-1]  |  A[i], A[i+1], ..., A[m-1]
B[0], B[1], ..., B[j-1]  |  B[j], B[j+1], ..., B[n-1]

这样，我们就将两个数列划分为两个部分，只要我们满足以下两个条件，就能找到中位数。

1 len(left_part) == len(right_part) (m+n为偶数)
len(left_part) == len(right_part) + 1 (m+n为奇数)
2 max(left_part) <= min(right_part)

即，i,j满足以下条件

1 if n >= m, j = (m + n + 1)/2 - i, i ∈ [0, m]
2 B[j-1] <= A[i] and A[i-1] <= B[j]

因此，寻找中位数的过程就简化为寻找一个合适的i值满足以上条件，我们可以通过二分法来达到O(log(min(m,n))的时间复杂度。

找到合适的i之后，由于我们之前的设定是m+n为奇数时，让left_part的长度大于right_part的长度，所以得到中位数：

max(A[i-1],B[j-1]), (当m+n为奇数时)
(max(A[i-1], B[j-1]) + min(A[i], B[j]))/2 (当m+n为偶数时)
